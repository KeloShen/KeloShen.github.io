My First Blog In My Life

发表于 2023-01-18 分类于 生活感慨 阅读次数：

前言

欢迎来到我的博客网站，希望通过这个网站可以给你带来帮助。

在偶然一次看到小红书评论的时候知道了可以通过博客来记录自己的学习笔记，并且可以对未来招聘也有帮助，所以花了一天的时间来搭建这个博客，希望自己未来能够经常在此记录自己的学习、情感、生活上的感悟，一直坚持努力下去。

如果你也想搭建自己的博客，可以观看这个视频进行学习

博客教程

Mac终端中常用的输入指令（基于hexo博客教程）

  • sudo su 切换root用户

  • clear 清理终端

  • node -v 查看当前版本

  • npm -v 同上

  • npm install -g cnpm –registry=http://registry.npm.taobao.org 安装

  • cnpm install -g hexo-cli 是hexo博客安装

  • hexo -v 检查

  • pwd 查看当前路径

  • mkdir 建立文件夹，后面加空格加文件夹名称

  • cd Blog/ 进入文件夹，左侧Blog为文件夹名称

  • cd .. 退出文件夹

  • cd ../.. 退出两层文件夹

  • sudo hexo init 用管理员的命令，初始化一个hexo博客

  • hexo s 启动你的博客，s为start的简写

  • hexo n “XXX我的第一篇博客文章XXX” 为生成一篇新的博客文章，n为new的简写

  • ls 查看当前文件夹中的文件

  • cd source/_posts/

  • ls -l

  • vim 我的第一篇博客文章.md 为编辑文章内容

  • 启用vim编辑后： 在英文输入状态中输入i，启用编辑功能；

    退出为esc+ ：wq

  • hexo四大步骤

    • hexo clean
    • hexo g （g为generate的简写）
    • hexo s （s为server的简写）
    • hexo d （d为deploy的简写）作用为推向远端

  • cnpm install –save hexo-deployer-git 安装

  • git clone https://example.com/abcdefg.git themes/yilia 下载主题到themes文件夹中
